kimi.rest API

Team radio and interview quotes by Finnish F1 legend Kimi Räikkönen

Usage

Construct a URI by appending an endpoint to the end of the website root domain

https://domain.com[endpoint]

See below for a list of available endpoints

Endpoints

GET a random quote

/quote

GET quote at a specified index

/quote/:index

GET all quotes

/quotes

GET all quotes from a specified year

/quotes/:year
